# Homework 4 — Java Streams

Functional programming tasks using Java Stream API.

## Tasks

1. **Odd-indexed names** — filter names at odd indices, format as `"1. Ivan, 3. Peter"`
2. **Uppercase & sort descending** — transform to uppercase, sort Z→A
3. **Extract & sort numbers** — parse numbers from string array, sort and join
4. **Linear congruential generator** — infinite stream via `Stream.iterate`
5. **Zip streams** — interleave two streams, stop when either is exhausted

## Run

```bash
javac src/main/java/homework/StreamTasks.java && java -cp src/main/java homework.StreamTasks
```
